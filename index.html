IZED is A Digital Archaeology project

Building cooperative network to preserve the cultural heritage worldwide, by collecting data from every resource available to start the cultural heritage digital documentation process.

Project goal:

Build a cooperative network to preserve the cultural heritage worldwide by create a platform that combines users from all over the world to participate by collecting data to build the cultural heritage documentation archive. like (images - videos - plans - textâ€¦..etc).

Interactive Timeline

The project IZED will start by sharing the timeline of the project with the users and take a part in editing the timeline and share their comments and suggestions besides choosing winch site we will start with and when.

Who can participate and help?

Everybody can help us to create this platform and any kind of informations are precious and important to the worldwide cultural preservation.

We have to collect as much as possible data we can.

Why now?

The Syrian war crisis was the main motivation to start this project due to the damage, abuse and loss that happened to many historical sites in Syria especially in Palmyra and Aleppo. This means that any historical site in the world is threatened to be lost or damaged in the future.

Why this project is so important?

With your help we will be able to preserve the cultural heritage by documenting all the collected informations in digital library that will be the reference for the next generations.
